1) Установить Git

2) Провести первоначальную настройку системы контроля версий git

3) Создать каталог для выполнения лабораторной работы(каталог проекта)

4) Инициализировать git

5) В каталоге проекта создать текстовый файл отчета (ФИО_ЛР1.txt в кодировке Win1251). В этом файле будет отчет по ЛР №1. Проиндексировать его.

6) Cделать первый коммит

7) Добавить в файл отчета описание всех действий которые вам пришлось выполнить до текущего момента.

8) Создали файл .gitignore в который записали папки и разрешение для игнорирования

9) Создали файл Delete и удалили его с помощью GitBash

10) Создали папку lib и переместили в нее, наш отчет

11) Воспользовались командой git log  и посмотрели историю коммитов
